Linked List Implementation in C

This repository contains implementations of Linked List data structures in C using two different approaches:
	1.	Array-Based Implementation
	2.	Dynamic Memory Allocation (Pointer-Based Implementation)

⸻

📌 Overview

Linked Lists are dynamic data structures that allow efficient insertion and deletion of elements. This project demonstrates two different ways to implement them:
	•	Array Implementation → Uses static memory with index-based linking
	•	Dynamic Implementation → Uses pointers and heap memory (malloc)

⸻

📂 Contents
	•	array_linked_list.c → Linked list using arrays
	•	dynamic_linked_list.c → Linked list using dynamic memory allocation

⚙️ Features Implemented

For both implementations:
	•	Create Linked List
	•	Traverse (Display elements)
	•	Insert:
	•	At Beginning
	•	At End
	•	At Any Position
	•	Delete:
	•	From Beginning
	•	From End
	•	From Any Position
	•	Find Length of Linked List

⸻

🧠 Concepts Covered
	•	Structures (struct)
	•	Pointers
	•	Dynamic Memory Allocation (malloc, free)
	•	Array simulation of linked list
	•	Memory management
